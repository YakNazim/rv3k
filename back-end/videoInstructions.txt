Video streaming

1.) Install avconv
	sudo apt-get install libav-tools
	
2.) Install nginx and nginx-rtmp module
	a. Install dependencies
		sudo apt-get install build-essential libpcre3 libpcre3-dev libssl-dev
	b. Download nginx and nginx-rtmp module
		wget http://nginx.org/download/nginx-1.9.15.tar.gz	
		wget https://github.com/arut/nginx-rtmp-module/archive/master.zip
		
	c. Unpack/unzip package
		tar -zxvf nginx-1.9.15.tar.gz
		unzip master.zip
		
	d. Go to nginx folder
		cd nginx-1.9.15
		
	e. Build and install:
		./configure --with-http_ssl_module --add-module=../nginx-rtmp-module-master
		make
		sudo make install
	
3.) Configure nginx.conf to set up RTMP
	a. open /usr/local/nginx/conf/nginx.conf with your editor and add the 
       following block at the very end of the file:
	
    rtmp {
            server {
                    listen 1935;
                    chunk_size 4096;

                    application live {
                            live on;
                            record off;
                    }
                    
                    application hls {
                            live on;
                            hls on;
                            hls_path /tmp/hls;

                    }
            }
    }


4.) Add this inside the http block
	
	location /hls {
		root /tmp;
		add_header 'Access-Control-Allow-Origin' '*';
	}
		
	

5.) Restart or start up nginx
    a.) To stop the nginx:
    	sudo /usr/local/nginx/sbin/nginx -s stop
    b.) To run:
    	sudo /usr/local/nginx/sbin/nginx
	
	
How to stream (example):

1.) Download launch video and stream
	wget http://annex.psas.pdx.edu/Launch-12/Video/360_1.mp4
	avconv -re -i 360_1.mp4 -c copy -f flv rtmp://<address>/hls/<stream name>
	
	
2.) Edit the manifest or rtmp url in rv3k web app (see ../front-end/ for instructions)
	m3u8 manifest file: http://<address>/hls/<stream name>.m3u8
	rtmp stream: http://<server>/hls/<stream name>
